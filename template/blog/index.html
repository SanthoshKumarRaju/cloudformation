About This Project

This blog documents the implementation of a production-ready 2-tier web architecture on AWS using Infrastructure as Code (CloudFormation). The architecture follows AWS best practices for security, scalability, and high availability.

  • VPC with public and private subnets
  • Application Load Balancer with Auto Scaling
  • RDS MySQL database in private subnets
  • Security groups for network isolation
  • Custom AMI with Apache web server
